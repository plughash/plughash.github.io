Command Arguments Description
/afk [message] Sets your AFK Message
/autoleave [number] Sets how many songs you'll play before leaving the waitlist.
/addvideo <YouTube URL> <Playlist Name> Adds a YouTube Video to the specified playlist.
/add <user> Adds the specified user to the waitlist.
/move <user> <position> Moves the specified user to the specified position.
/remove <user> Removes the specified user from the waitlist.
/skip Force Skips the current DJ.
/ban <user> Permanently bans the specified user.
/kick [hour|day] Kicks the user, if no parameter is specified its a day ban.
/clearchat Clears the chat, sends message afterwards.
/delete <user> Deletes messages of the specified user.
/link Sends a link to the current song.
/whois [username|id] Returns information about the specified user.
/reload Reloads plug#.
/kill Shuts down plug#.
